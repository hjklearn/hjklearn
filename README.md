I am an active researcher focusing on developing efficient network architectures, **knowledge distillation**, and **few-shot learning**. My goal is to solve complex visual recognition challenges across both **medical imaging** and **industrial inspection** domains.

---

### 🔬 Research Interests

- 🩺 **Medical Image Segmentation:** Few-Shot Learning, Knowledge Distillation *(e.g., Infusion to Assimilation paradigms)*.
- 👁️ **Computer Vision:** Object Detection, Image Classification, Interactive/Elastic Prototype Networks.
- 🏭 **Industrial Defect Inspection:** RGB-D Scenes, Real-Time Processing for Rail Surface Defect Detection.
